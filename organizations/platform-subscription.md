# Platform Subscription

Organizations that manage money on Open Collective subscribe to a platform plan based on their activity level. Your plan determines which features are available (payout integrations, expected funds, bank sync, and more).

## View your plan

1. Switch to your Organization in the dashboard account switcher.
2. Navigate to **Settings > Platform Billing**.

This page shows your current tier, monthly price, included collectives and expenses, and usage against those limits.

## Choose or change a plan

From Platform Billing you can compare **Discover**, **Basic**, and **Pro** tiers and select a package that matches your active collectives and monthly expense volume.

Each tier includes:

* A monthly subscription price
* A number of **active collectives** included (with a per-collective overage rate)
* A number of **paid expenses per month** included (with a per-expense overage rate)
* A feature set (see [Pricing](../why-open-collective/pricing.md) for the tier comparison)

Use **See more plans** on the public [pricing page](https://opencollective.com/organizations/pricing) to compare package sizes within a tier.

## Feature access by tier

| Feature | Discover | Basic | Pro |
| ------- | :------: | :---: | :-: |
| Crowdfunding pages | Yes | Yes | Yes |
| Expense management | Yes | Yes | Yes |
| Chart of Accounts | No | Yes | Yes |
| Payouts with Wise / PayPal | No | Yes | Yes |
| Expected funds | No | Yes | Yes |
| Charge hosting fees | No | Yes | Yes |
| Antifraud security checks | No | Yes | Yes |
| Bank Account Synchronization | No | No | Yes |
| Tax Forms | No | No | Yes |
| Funds and Grants | No | No | Yes |
| Agreements | No | No | Yes |

If you need a feature that is not on your current tier, upgrade from Platform Billing.

## Migration from legacy pricing

Organizations on the previous pricing model (free organization access or 15% platform share on host fees) may see a banner prompting migration to the new subscription model. Legacy pricing details are at [opencollective.com/legacy-pricing](https://opencollective.com/legacy-pricing).

## Crowdfunding fee model

By default, crowdfunding uses optional [Platform Tips](../giving-to-collectives/platform-tips.md). Organizations can request a **5% Crowdfunding Fee** instead. [Contact us](https://opencollective.com/contact) to switch.
